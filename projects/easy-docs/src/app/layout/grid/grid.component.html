Grid

In order for grid to work, a parent container must have a .grid class.

Because of the complexity and flexibility of grid, EASY only provides simple styles for layout.

For named areas and lines, please provide your own styles. See A Complete Guide to Grid for more details.

Container Columns and Rows

Use .cols-[1 - 12] and .rows-[1 - 12] classes on the grid container to build your grid.

Container Column and Row Gaps

Use .col-gap-[xs || sm || md || lg || xl] and .row-gap-[xs || sm || md || lg || xl] classes on the grid container to add grid gaps.

A gap-[xs || sm || md || lg || xl] class can be used to set both.
