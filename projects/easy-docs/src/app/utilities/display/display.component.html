Display

Display classes specify how an element should behave.

Block

Use a .dis-b class to override an inline or inline block element.

This makes the element full width with the ability to set the height and width.

block

Inline

Use a .dis-i class to override a block or inline block element.

This makes the element fit the content with no ability to set the height and width.

inline

Inline Block

Use a .dis-i-b class to override a block or inline element.

This makes the element fit the content with the ability to set the height and width.
