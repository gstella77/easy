Misc

Several miscellaneous helper classed have been added for convienence.

Lists

Because <ul>s, <ol>s, and <li>s are often used for document structure, all formatting has been removed.

If you want them to represent list items, use the .list class on <ul>s and <ol>s.

  1. item 1
  2. item 2
    1. sub item 1
    2. sub item 2

Center

Elements can be centered with a .center class.
